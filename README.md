# Zomato Dataset Analysis

This folder contains the completed Alfido Tech internship Task 1 submission for **Zomato Dataset Analysis**.

## Files

- `zomato_analysis.ipynb` - Jupyter/Colab notebook with data cleaning, EDA, visualizations, word cloud, and recommendations.
- `zomato_report.md` - Report version of the findings and recommendations.
- `requirements.txt` - Python packages needed to run the notebook locally.

## How to Run as a Beginner

1. Open [Google Colab](https://colab.research.google.com/).
2. Upload `zomato_analysis.ipynb`.
3. Run the first setup cell.
4. When prompted, upload your Kaggle API key file named `kaggle.json`.
5. Run all cells from top to bottom.
6. In Colab, export the notebook/report as PDF using:
   - `File` -> `Print` -> `Save as PDF`, or
   - `File` -> `Download` -> `Download .ipynb` for the notebook deliverable.
